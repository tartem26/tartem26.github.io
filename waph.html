Overview

About This Course

Web Application Programming and Hacking is a hands-on course focused on web foundations, front-end development, server-side programming, cloud deployment, and web security. The course connects practical web development with security concepts such as input validation, output encoding, Cross-Site Scripting, SQL, cookies, and sessions.

This page documents the related hands-on projects, labs, hackathons, and Individual Project 1 work completed for the course.

Individual Project 1

Website Features

  • Professional profile homepage with resume content, contact information, skills, experience, education, awards, projects, community activities, and hobbies.
  • Open-source CSS framework through Bootstrap.
  • JavaScript features including digital clock, analog clock, show/hide email, and theme toggle.
  • jQuery and Day.js as open-source JavaScript libraries.
  • JokeAPI integration with a new joke every 1 minute.
  • Dog CEO image API integration with a random dog image.
  • JavaScript cookies to remember the visitor and show the last visit date/time.
  • Third-party page tracker shown on the homepage.
